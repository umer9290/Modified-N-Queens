---------------------------------------------------
Observations / Modifications for Larger N:
---------------------------------------------------

BFS expands nodes level by level without considering cost or heuristic, 
which leads to large expansion for higher N.

UCS expands based on cumulative path cost. Since each move has uniform 
cost in this problem, UCS behaves similarly to BFS but is implemented 
using a priority queue.

A* performs significantly better for larger N because it uses a heuristic 
(conflict count) to guide the search toward promising states.
